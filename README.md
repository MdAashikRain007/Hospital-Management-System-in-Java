# 🏥 Hospital Management System (Java Console Application)

A Java console-based Hospital Management System designed to manage patients, doctors, and appointments efficiently using JDBC and MySQL.

---

## 📌 Project Description

This project is a backend console application built using **Core Java** and **JDBC**.  
It connects to a MySQL database to perform CRUD operations for hospital management.

The system allows hospital staff to:

- Add and manage patients
- View available doctors
- Book appointments
- Store and retrieve records from the database

---

## 🚀 Features

- 👨‍⚕️ Add New Patients
- 📋 View Patient Details
- 🩺 View Doctor List
- 📅 Book Appointments
- 🗄 Database Connectivity using JDBC
- 💾 MySQL Integration

---

## 🛠️ Technologies Used

- Java (Core Java)
- JDBC (Java Database Connectivity)
- MySQL Database
- IntelliJ IDEA 
- MySQL Connector/J
